# Ray Train 微调 Qwen3.5-0.8B

## 📋 项目说明

使用 Ray Train + Hugging Face Transformers 微调 Qwen3.5-0.8B 模型。

## 🚀 快速开始

### 1. 安装依赖（首次运行）

```bash
conda activate aaa
pip install -r requirements.txt
```

### 2. 一键训练

```bash
python main.py
```

或使用脚本：

```bash
bash run.sh
```

## 📁 文件结构

```
Ray-train/
├── main.py              # 主训练脚本
├── requirements.txt     # Python 依赖
├── run.sh              # 运行脚本
├── README.md           # 说明文档
└── checkpoint_final/   # 训练完成后的模型输出目录
```

## ⚙️ 配置说明

在 `main.py` 中的 `TrainingConfig` 类可以调整以下参数：

### 路径配置
- `model_path`: 基础模型路径（默认：`../Qwen3.5-0.8B`）
- `train_data_path`: 训练数据路径（默认：`../data/train_dataset.json`）
- `val_data_path`: 验证数据路径（默认：`../data/val_dataset.json`）
- `output_dir`: 输出模型路径（默认：`./checkpoint_final`）

### 训练超参数
- `num_train_epochs`: 训练轮数（默认：3）
- `per_device_train_batch_size`: 每设备训练批次大小（默认：4）
- `gradient_accumulation_steps`: 梯度累积步数（默认：4）
- `learning_rate`: 学习率（默认：2e-5）
- `max_seq_length`: 最大序列长度（默认：2048）

### 优化设置
- `fp16`: 使用混合精度训练（默认：True）
- `gradient_checkpointing`: 梯度检查点（默认：True，节省显存）

## 💾 模型加载

训练完成后，使用以下代码加载微调后的模型：

```python
from transformers import AutoModelForCausalLM, AutoTokenizer

model = AutoModelForCausalLM.from_pretrained(
    "./checkpoint_final",
    trust_remote_code=True,
    device_map="auto"
)

tokenizer = AutoTokenizer.from_pretrained(
    "./checkpoint_final",
    trust_remote_code=True
)

# 推理示例
messages = [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "Hello!"}
]

text = tokenizer.apply_chat_template(
    messages,
    tokenize=False,
    add_generation_prompt=True
)

inputs = tokenizer([text], return_tensors="pt").to(model.device)
outputs = model.generate(**inputs, max_new_tokens=512)
response = tokenizer.decode(outputs[0], skip_special_tokens=True)
print(response)
```

## 🔧 故障排除

### 显存不足（OOM）
如果遇到显存不足，可以调整以下参数：
- 减小 `per_device_train_batch_size`（如改为 2 或 1）
- 增加 `gradient_accumulation_steps`（保持有效批次大小）
- 减小 `max_seq_length`（如改为 1024）

### CUDA 错误
确保已安装正确的 PyTorch CUDA 版本：
```bash
python -c "import torch; print(torch.cuda.is_available())"
```
